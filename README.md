# Cancer Staging — quick reference

An offline, installable web-app summarising solid-tumour staging (AJCC 8th/9th,
FIGO, and disease-specific systems such as BCLC, Lugano, R-ISS, WHO CNS5).

**For clinicians and medical education.** A navigational reference summarising
published staging systems — verify against the current AJCC manual, FIGO statement
or disease-specific source before any clinical use. Not medical advice or a
substitute for professional judgement. Staging editions change.

## Install on iPhone/iPad
Open the site in **Safari** → **Share → Add to Home Screen**. It then runs
full-screen and works offline.

Generated from `staging_data.json` via `build_staging_dashboard.py`.
